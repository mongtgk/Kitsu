# Документация Kitsu

 Kitsu — открытый сервис для каталога аниме и HLS‑воспроизведения. Вся актуальная документация находится в этой папке и описывает только текущее состояние системы.

> Папка `contracts/` — обязательная точка истины. Перед любыми изменениями кода агент обязан прочитать соответствующие контракты. Любой документ вне `contracts/`, который противоречит контракту, считается устаревшим и требует обновления.

## Структура
- `contracts/` — обязательные архитектурные контракты (auth, api-boundary, background-jobs, rbac, frontend-backend).
- `ARCHITECTURE.md` — целостное описание компонентов и потоков данных.
- `BACKEND.md` — API FastAPI, зависимости, ошибки, фоновая обработка записей.
- `FRONTEND.md` — Next.js UI, источники данных, RBAC в интерфейсе.
- `BACKGROUND_JOBS.md` — очередь фоновых задач и текущие задания (приоритет у `contracts/background-jobs.md`).
- `RBAC.md` — роли, разрешения и способы применения (приоритет у `contracts/rbac.md`).
- `RBAC_CONTRACTS.md` — соответствие эндпоинтов и требуемых разрешений.
- `GLOSSARY.md` — термины и определения.
